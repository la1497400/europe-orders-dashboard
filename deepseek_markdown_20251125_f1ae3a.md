# 欧洲本土订单管理系统

这是一个基于 Supabase 和 Netlify 的订单管理系统，用于查看和管理欧洲本土的电商订单数据。

## 功能特点

- 📊 实时数据展示
- 🔍 高级筛选和搜索
- 📈 销售统计和图表
- ✏️ 在线编辑订单状态
- 📱 响应式设计

## 技术栈

- 前端: HTML, CSS, JavaScript, Bootstrap
- 后端: Supabase (PostgreSQL)
- 部署: Netlify

## 本地运行

1. 克隆或下载此项目
2. 使用浏览器打开 `index.html` 文件

## 部署

此项目已配置为可直接部署到 Netlify。

## 数据源

数据来自 Supabase 数据库表 "欧洲本土"。
